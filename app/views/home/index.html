Welcome to WUR

WUR is the acronym for World University Ranking.
This application displays the results yielded by a study carried out using university data from Kaggle.
Information is available in both tabular or graphical forms, which provides users with different vantage points with regard to data exploration.
Hopefully you will have fun while learning more about universities around the world.
